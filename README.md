# Insites Ecommerce Template
Latest `v5.0.0`

## Overview
The Ecommerce Template provides a starting point in building the ecommerce part of the website that includes product listing, product details, shopping cart, checkout process up to payment. This module also includes the base for customer account creation like sign-up, sign-in, and account information.

This template requires Insites Website Template to be installed first.

The Ecommerce Template is stored in the public folder of the model, “modules/ins_template_ecommerce/..”.

## Tech Stack
It is assumed that the developers / users have knowledge of using these:

 - PlatformOS
 - Insites Components and Modules
 - Liquid
 - GraphQL
 - YAML
 - Javascript
 - CSS / SCSS
 - HTML
